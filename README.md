# Mockups Usabilidad

Este es un proyecto construido con Next.js que utiliza `pnpm` como gestor de paquetes.

## Cómo correr el proyecto pasito a pasito (Local)

Sigue estas instrucciones para ejecutar el entorno de desarrollo y ver el proyecto en vivo en tu computadora.

### Requisitos Previos
Asegúrate de tener instalado [Node.js](https://nodejs.org/) y `pnpm`.
Si no tienes `pnpm`, puedes instalarlo ejecutando este comando en tu terminal:
```bash
npm install -g pnpm
```

### Paso 1: Abrir la terminal
Abre una terminal (Símbolo del sistema, PowerShell o la terminal integrada de tu editor de código, como VS Code) y asegúrate de estar en la raíz de la carpeta del proyecto. Por ejemplo:
```bash
cd "c:\Users\Juan Felipe Hurtado\Documents\mockups-usabilidad"
```

### Paso 2: Instalar las dependencias
Para descargar e instalar todas las dependencias y librerías que requiere el proyecto, ejecuta el siguiente comando:
```bash
pnpm install
```
*(Si no tienes `pnpm` instalado, puedes instalarlo primero corriendo `npm install -g pnpm`).*

### Paso 3: Iniciar el servidor de desarrollo
Una vez que las dependencias estén completamente instaladas, levanta el proyecto en modo de desarrollo con el siguiente comando:
```bash
pnpm run dev
```

### Paso 4: Ver el proyecto en el navegador
Después de ejecutar el comando anterior, la terminal te indicará que el servidor está corriendo (usualmente en el puerto 3000). 
Abre tu navegador web de preferencia y dirígete a la siguiente dirección:

[http://localhost:3000](http://localhost:3000)

Ahí podrás ver y probar el proyecto en vivo. Cualquier cambio que realices en el código se reflejará y recargará automáticamente en la página.
